Learn the fundamentals

A quadratic equation is any equation containing a second‑degree polynomial. In its standard form it can be written as ax2 + bx + c = 0 where a, b and c are real numbers and a ≠ 0【421980112223373†L41-L88】. Quadratic equations appear throughout science and engineering.

The graph of the quadratic function f(x) = ax2 + bx + c is a parabola. Its shape depends on the coefficients: if a > 0 the parabola opens upward and has a minimum point; if a < 0 it opens downward and has a maximum. The extreme point of the parabola is called the vertex【78706179724552†L474-L483】.

Vertex & axis of symmetry: The x‑coordinate of the vertex is given by −b∕(2a) and the y‑coordinate is found by substituting this value back into the function【78706179724552†L474-L483】. The vertical line through the vertex is called the axis of symmetry. The y‑intercept of the graph occurs at (0, c)【78706179724552†L474-L485】.

Discriminant: When solving ax2 + bx + c = 0 with real coefficients, the expression under the square root in the quadratic formula is called the discriminant Δ = b2 − 4ac. The discriminant determines the number and nature of the roots: if Δ > 0 the equation has two distinct real roots; if Δ = 0 there is exactly one real double root; and if Δ < 0 the roots are complex conjugates【78706179724552†L440-L460】.

Quadratic formula: Any quadratic equation can be solved using the quadratic formula \(x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)【421980112223373†L365-L378】. The discriminant appears under the square root.

Factoring method: When a quadratic can be written as a product of two linear terms, the zero‑product property allows us to set each factor equal to zero. Factoring works by finding two numbers whose product equals c and whose sum equals b; after writing the factors, setting them equal to zero gives the solutions【421980112223373†L41-L54】.

Vertex form: A quadratic can also be written in vertex form as a(x − h)2 + k, where (h, k) is the vertex of the parabola. Converting between standard and vertex form is useful for graphing and describing transformations. Expanding a(x − h)2 + k gives ax2 − 2ahx + (ah2 + k).
